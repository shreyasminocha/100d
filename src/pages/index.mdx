# 100D Political Model

Based on [Jreg](//youtube.com/channel/UCGSGPehp0RWfca-kENgBJ9Q)'s [_The 100-dimensional political model_](//youtube.com/watch?v=UuopBeaUN24). A lot of this won't make sense if you haven't watched the video.

Near-replica of [u/DontDoMarmalade](https://www.reddit.com/user/DontDoMarmalade)'s [axis&nbsp;chart](https://www.reddit.com/r/Jreg/comments/djd0qx/i_made_an_actual_axis_chart_for_the_100/).
